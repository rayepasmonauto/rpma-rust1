# 05. IPC API AND CONTRACTS

The IPC (Inter-Process Communication) layer is the bridge between the React frontend and the Rust backend.

## Contract Rules
- **Response Envelope**: All responses are wrapped in an `ApiResponse<T>` object:
  ```typescript
  {
    success: boolean;
    data?: T;
    error?: { code: string; message: string };
    correlation_id: string;
  }
  ```
- **Auth Token**: Handled by the Tauri state and `resolve_context!` macro.
- **Correlation ID**: Generated by the frontend or backend to trace a request through its entire lifecycle.

## Type Synchronization
Rust models are exported to TypeScript using the `ts-rs` crate.
- **Rust Source**: `#[derive(TS)] #[ts(export)]` on structs/enums.
- **Generated Path**: `frontend/src/types/` (via `src-tauri/bindings/`).
- **Sync Command**: `npm run types:sync`.

## Top 10 Most Important Commands

| Command | Domain | Purpose | Permissions |
| --- | --- | --- | --- |
| `auth_login` | Auth | Authenticate user and start session | Guest |
| `task_crud` | Tasks | Create, Read, Update, Delete tasks | Role-dependent |
| `client_crud` | Clients | Manage client records | Role-dependent |
| `intervention_start` | Interventions | Initialize a technical workflow | Technician+ |
| `intervention_advance_step` | Interventions | Move to the next step in a workflow | Technician+ |
| `material_list` | Inventory | Get available materials and stock levels | Viewer+ |
| `material_record_consumption`| Inventory | Record materials used during work | Technician+ |
| `get_events` | Calendar | Fetch scheduled events for a range | Viewer+ |
| `quote_create` | Quotes | Generate a new work estimate | Supervisor+ |
| `report_generate` | Documents | Generate an intervention PDF report | Technician+ |
